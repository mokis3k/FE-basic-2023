Git commands I know:

  1. git add: команда для початку відстеження файлів (те, що треба робити перед комітом)
  2. git branch: створення гілок
  3. git checkout: переключення між гілками
  4. git clone: клонування вже створенного репозиторію
  5. git commit: фіксує поточний стан коду в "історію" проєкту
  6. git push: публікація гілки на сервер
  7. git pull: завантаження гілки з серверу
  8. git remote: скорочення для повної url адреси
  9. git status: показує поточний стан робочої області
  10. git fetch: завантаження файлів та всіх комітів з віддаленого репозиторію, до локального

    11. *Взагалі щодо останньої команди. Я не пам'ятаю чи я її почув на уроці, чи згадав з попередньго курсу
