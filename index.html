css selectors

  • class - this selector selects elements based on the value of their attribute "class".
    syntax:
    .class-name {}
  • id - this selector selects elements based on the value of their attribute "id".
    syntax:
    #id-name {}
  • attr - this selector selects all elements that have the given attribute or an attribute with a specific value.
    syntax:
    [attr=value] {}
  • tag/element - this selector selects the type of elements that the rule will apply to.
    syntax:
    span {}
  • " * " - selects all elements. optionally, it can be limited to a specific namespace or refer to the entire namespace.
    syntax:
    * {}
  • " , " - grouping method, it selects all matching selectors.
    syntax:
    div, span {}
  • " " - selects elements that are inside the specified element (regardless of nesting level).
    syntax:
    div span {}
  • " > " - selects only those elements that are directly children of the specified element.
    syntax:
    ul > li {}
  • " ~ " - selects elements that are at the same nesting level, after the specified element, with the common parent.
    syntax:
    p ~ span {}
  • " + " - selects the element immediately after the specified element if they have a common parent.
    syntax:
    ul + li {}
